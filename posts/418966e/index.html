VSCode 快捷键大全

常用快捷键

单行注释

Ctrl + /

多行注释

Alt + Shift + A

关闭当前窗口

Ctrl + W

关闭所有窗口

Ctrl + K + W

选中字母

Shift + ↑ / ↓ / ← / →

选中单词

Ctrl + Shift + ↑ / ↓ / ← / →

移动行

Alt + ↑ / ↓

刪除行

Shift + Ctrl + K

自动换行

Alt + Z

格式化代码

Shift + Alt + F

新建一个窗口

Ctrl + Shift + N

在文件中查找

Ctrl + Shift + F

在文件中替换

Ctrl + Shift + H

快速打开文件

Ctrl + P

编辑器命令面板

Ctrl + Shift + P

打开最近的文件

Ctrl + R

快速打开终端

Ctrl + ~

快速打开和关闭侧边栏

Ctrl + B

辅助快捷键

清除多余空格

Ctrl + K Ctrl + X

折叠代码

Ctrl + Shift + [

展开代码

Ctrl + Shift + ]

往上或下复制代码

Shift + Alt + ↑ / ↓

跳转到特定行数

Ctrl + G

跳转到文件中的特定符号

Ctrl + Shift + O

跳转到项目中的特定符号

Ctrl + T

删除整个单词

Ctrl + Backspace

多行选中同时编辑

Shift + Alt + 鼠标选中

查看全部快捷键

Ctrl + K + S

编辑器网格布局

打开方式：查看 > 编辑器布局 > 2x2 网格
